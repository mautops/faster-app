# 如何贡献

感谢你考虑为 Faster APP 做出贡献！

## 贡献方式

### 报告 Bug

在 [GitHub Issues](https://github.com/mautops/faster-app/issues) 提交 Bug 报告。

**请包含**：

- 操作系统和 Python 版本
- Faster APP 版本
- 重现步骤
- 期望行为和实际行为
- 错误日志

### 提交 Pull Request

1. Fork 本仓库
2. 创建特性分支：`git checkout -b feature/amazing-feature`
3. 提交更改：`git commit -m 'Add amazing feature'`
4. 推送分支：`git push origin feature/amazing-feature`
5. 提交 Pull Request

## 代码审查

所有提交都需要经过代码审查才能合并。
